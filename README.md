# Skill Quick Start

Copy `ai-spec-coding-framework/` into your Copilot skills folder.

Keep the folder name `ai-spec-coding-framework`, and keep `SKILL.md` at the top of that folder.

Open Copilot, start the skill, and answer these three prompts:

1. Project name
2. Programming language and tech stack
3. Project background and keywords

After the third answer, the skill generates the initial framework: `.ai/`, `doc/specs/`, `src/`, and `test/`.
